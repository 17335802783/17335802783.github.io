java数据类型

1.基本数据类型

​ 1.1 整型

  • byte 1字节=8bit位 范围：-128——127 2^8=256

  • short 2字节

  • int java整型默认int 4字节 32位

  • long 8字节

    声明long变量时，如果值超过int，那么需要在值后面加L bit是计算机最小存储单位，byte是基本存储单元。

​ 1.2 浮点型

  • float 4字节 单精度

  • double 8字节 双精度 java浮点型 默认 double

    声明float 需在后面加 “f “/ “F”

​ 1.3 字符型

  • char 1字符=2字节
  • 定义char变量，通常使用一对 ‘ ‘ ,内部仅有一个字符
  • 可以赋值转义字符， Unicode 值表示字符型常量

​ 1.4 布尔型

  • boolean 1位（底层存储为0/1）

  • 取值 “ true” , “ false”
